We Are Data Scientists

This web app was made for our Data 301 final project at California Polytechnic State University. We wanted to work with the large datasets provided by Yelp in JSON formatto derive some interesting inferences. The two main componenets consist of using Natural Language Processing to derive ratings from reviews and using Sentiment Analysis to construct word clouds.
